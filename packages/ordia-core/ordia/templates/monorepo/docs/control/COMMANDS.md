# Commands

Project command overlay (L2/L3). L1 npm scripts live in `package.json`.

- Catalog: `docs/control/commands.catalog.json`
- Ordia pip CLI spec: `docs/ordia/package/COMMANDS.md` (when installed with `--with-docs`)

Validate: `ordia validate --project` (primary). Optional npm wrapper: `npm run ordia:validate` when `package.json` scripts are wired.
